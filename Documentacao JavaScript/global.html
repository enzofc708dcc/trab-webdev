ajax(url, method, callback, body)

É responsável pela comunicação do frontend com o backend, chamando um script php para executar uma consulta ao banco de dados. A resposta dessa consulta é então consumida por uma função do frontend.
Parameters:
Name Type Default Description
url String url para o código php a ser executado.
method String método HTTP a ser executado.
callback function função a ser executada com a resposta.
body String null corpo da requisição caso método HTTP seja POST.
